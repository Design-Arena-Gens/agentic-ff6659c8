## Passionate Performer Stage Illustration

This project is a minimal Next.js experience that renders a cinematic illustration of a young Indian performer singing into a vintage microphone beneath blue concert lighting.

### Tech Stack
- Next.js 14 (App Router)
- React 18
- Hand-crafted SVG for the illustration

### Getting Started
```bash
npm install
npm run dev
```
Then visit `http://localhost:3000`.

### Production Build
```bash
npm run build
npm start
```

### Deployment
Deploy to Vercel with:
```bash
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-ff6659c8
```
